# Nimbus — SaaS Landing Page (Demo)

A modern, fully responsive SaaS / startup landing page built with plain
HTML, CSS, and JavaScript. No build step, no dependencies. Perfect as a
portfolio piece and Fiverr gig sample.

## Sections
- Sticky navbar with mobile menu
- Hero with animated dashboard mockup + floating cards
- Trusted-by logo bar
- Features grid (6 cards)
- How it works (3 steps)
- Stats band
- Pricing (3 tiers, highlighted "popular" plan)
- Testimonials
- FAQ accordion
- Email capture call-to-action
- Footer

## Run locally
Just open `index.html` in any browser. That's it.

## Deploy free (and get a live link for your CV / Fiverr gig)

### Option A — GitHub Pages (easiest)
1. Push this repo to GitHub (`main` branch).
2. Go to **Settings → Pages**.
3. Source: `Deploy from a branch` → branch `main` → folder `/ (root)` → Save.
4. Live in ~1 minute at: `https://pasindu-cyber-cloud.github.io/web/`

### Option B — Netlify / Vercel
- Drag-and-drop the folder at [netlify.com/drop](https://app.netlify.com/drop), or
- Import the repo at [vercel.com](https://vercel.com) → Deploy.

## How to capture a gig image
1. Open the live site (or `index.html`) in your browser.
2. Take a full-page screenshot, or screenshot the hero section.
3. Crop to **1280 x 769 px** (Fiverr's recommended gig image ratio).
4. Upload to your Fiverr gig Gallery.

## Customize
- Brand name / copy: edit text in `index.html`
- Colors: change the CSS variables at the top of `styles.css` (`--brand`, `--brand-2`, etc.)
